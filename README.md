# Spreadsheet Clone

An interactive HTML-based spreadsheet clone that allows users to create and manipulate a grid of data. This project is built primarily using HTML, CSS, and JavaScript, and it includes features such as text formatting, data manipulation, and basic statistics computation.

## Features

- Editable grid of cells with A1 notation addressing.
- Supports text formatting (bold, italic, underline).
- Data manipulation functions (trim, convert to upper case, lower case, remove duplicates).
- Find and replace functionality.
- Displays statistics (sum, count, average, min, max) of the numeric values in the grid.
- Ability to change font and cell colors.
- Export grid data to a JSON file.
